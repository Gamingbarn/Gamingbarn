This datapack allows players to morph into several different mobs.
You can morph by running the command:
/trigger morph
Alternatively, you can morph via:
/trigger [MORPH_MOB]
Replace [MORPH_MOB] with the mob you wish to morph into.
Up to 5 players can be the same morph at once.
----------------------------------------------
You can change the variant of certain morphs by running the command:
/trigger skins
Alternatively, you can run the command to change the mob and variant directly:
/trigger [MOB]_[VARIANT_NAME]
Replace [​MOB] with the name of the mob and [​VARIANT_NAME] with the name of the variant.

If you want to disallow certain player's abilities to morph you can run the command:
/tag PLAYER_NAME add no_morph
If you would like to give back their ability to morph, run the command:
/tag PLAYER_NAME remove no_morph
