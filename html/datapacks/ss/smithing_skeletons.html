---------------------------------------------------------------------------------------------
This datapack adds 2 new mobs to the game.
The Toolsmith Skeleton and the Amorsmith Skeleton.
If a skeleton can spawn, then a toolsmith or armorsmith can spawn.
These 2 skeletons can drop unbreakable iron tools and armor.
The toolsmith will drop any unbreakable iron tool.
While the armorsmith will drop any unbreakable iron armor.
It is completely random which item they drop.
---------------------------------------------------------------------------------------------
You can summon the mobs this datapack adds with the command:
/function summon:[NAME_OF_MOB]
Replace [NAME_OF_MOB] with the name of the mob you want to summon.
