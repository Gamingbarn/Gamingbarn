When you see a regular zombie while playing Minecraft, you probably think nothing of it.
But when you see one of these zombies...
You'll run.
--------------------------------------------------------------------------------------------
This datapack completely changes the spawning mechanic for mobs.
No more non-zombie hostile mobs in the overworld.
Most hostile mobs will now spawn as zombies.
All zombies will spawn with more health, more attack damage, and faster speed.
But it doesn't end there.
There is a chance that pigs, cows, sheep, chickens, or even horses will spawn as zombified variants.
HOWEVER!
Zombies are very weak to fire!
They will die very quickly if on fire.
All zombies will also leave a trail of fire when they are on fire.
--------------------------------------------------------------------------------------------
Don't die.
Because dying will infect you with the zombie virus.
As a zombie you are slightly stronger and zombies will no longer attack you.
However you have slowness and mining fatigue 2, making it very difficult to break blocks.
So go cure yourself fast.
But rumor has it that some zombies have even tried to cure themselves.
So they might have a special item on them when they die.
--------------------------------------------------------------------------------------------
Regular Zombie
-Regular Zombie
Roamer Zombie
-Is faster but has less health.
Nurse Zombie
-Drops gunpowder.
(When you have Gamingbarn's Guns installed they will drop various medical items.) (Even cures.)
Killer Zombie
-Wields a special sword that it can drop on death.
-Because it has a sword it does more damage.
Soldier Zombie
-Has resistance 1.
(When you have Gamingbarn's Guns installed they will drop more ammo.)
Scorched Zombie
-Is immune to fire.
(When you have Gamingbarn's Guns installed they will drop more ammo.)
Swarm Zombie
-If you find one, there is always more nearby.
Spider Zombie
-Spawns from cave spider spawners.
-Drops spider eyes.
Builder Zombie
-Will place blocks on the ground level around itself
Miner Zombie
-Will break blocks near itself
(note: will only break simple blocks like wood and obblestone)
--------------------------------------------------------------------------------------------
One more thing to mention.
Zombies will call for reinforcements if they are hurt.
So be careful.
--------------------------------------------------------------------------------------------
This datapack also has some commands players with OP can run
Run the following commands to turn off/on player infections. (This is on by default)
/function config:infection_on
Run the following commands to turn off/on zombie animals. (This is on by default)
/function config:animal_zombies_off
/function config:animal_zombies_on
Run the following commands to turn off/on zombie builders/miners. (This is on by default)
/function config:special_zombies_off
/function config:special_zombies_on
Run the following command to summon any of the custom zombies
(Change [ZOMBIE] with the type ofzombie)
/function summon:[ZOMBIE]
